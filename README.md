### Tech stack:
 - ExressJs
 - Docker
 - Sequelize
 - PostgreSQL
 - RabbitMQ
 

### Run the full stack using:

```
docker-compose up --scale pub=2
```

### Api Endpoint:

| Route | Description | 
|:-----------:|:------------:|
| /api/average | Returns the average of stored number for the last 5 seconds |
